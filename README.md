### Installation

Clone the repository

```bash
git clone https://github.com/MaheenGitHub/redefining-love.git
```
Go into the project folder

```bash
cd redefining-love
```

Install dependencies
```bash
npm install
```

Run development server
```bash
npm run dev
```

Build for production
```bash
npm run build
```

Preview production build
```bash
npm run preview
```
